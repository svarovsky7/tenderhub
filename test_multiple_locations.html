Пример Excel файла с множественными локализациями

Пример того, как должен выглядеть Excel файл для импорта одного вида затрат с несколькими локализациями:

A: № B: Категория C: Ед.изм. категории D: Вид затрат E: Ед.изм. детали F: Локализация
1 Отделочные работы м²
Отделка полов м² Автостоянка
Отделка полов м² Технические помещения
Отделка полов м² МОП ПЧ
Покраска стен м² Офисы
Покраска стен м² Коридоры

Результат импорта:

  • Категория: Отделочные работы
  • Детализация "Отделка полов" будет создана 3 раза с разными локализациями:
    • Отделка полов → Автостоянка
    • Отделка полов → Технические помещения
    • Отделка полов → МОП ПЧ
  • Детализация "Покраска стен" будет создана 2 раза:
    • Покраска стен → Офисы
    • Покраска стен → Коридоры

Изменения в логике импорта:

  1. Проверка существующих записей теперь учитывает не только имя детализации и категорию, но и локализацию
  2. Один вид затрат может иметь множество записей с разными локализациями
  3. В интерфейсе детализации группируются по имени с показом количества локализаций
